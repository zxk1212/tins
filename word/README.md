# Public corpus sources for InterNeg

- `common20k_20k.txt`: downloaded from `first20hours/google-10000-english` raw `20k.txt`.
- `part_of_speech_370k_words_alpha.txt`: downloaded from Hugging Face dataset `MB-IDK/EN-wordfreq-370k` raw `words_alpha.txt` (370,105 words).
- `part_of_speech_370k_nltk_tagged.csv`: locally reconstructed POS CSV with columns `index,word,pos_tag`, generated by applying NLTK Penn-Treebank tagging to `part_of_speech_370k_words_alpha.txt`.

Note: the paper/NegLabel footnote points to the Kaggle 370K POS corpus, but that direct file was not fetchable from this environment. The generated CSV is a machine-local reproducible substitute built from a public 370K word source and NLTK POS tagging.
